## 📖 介绍

1. 这是一个用来给 Stable Diffusion 写关键词用的小工具
2. 只需要在网页下面选择对应的关键词就可以汇总到上方的框框中，可以选择复制或清空。
3. 可以对关键词以Stable Diffusion的权重规则添加或减少权重比！-> ` “()” ` 和 ` “[]” `。
